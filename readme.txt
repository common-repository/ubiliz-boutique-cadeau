Ce plugin E-commerce vous permet d'afficher la liste de vos bons cadeaux sur votre site Internet Wordpress. Vendez et gérez vos bons cadeaux en ligne simplement et en quelques clics avec le plugin de la solution ecommerce Ubiliz.

== Description ==

Intégrez une boutique commerce de vente de bons cadeaux, cartes cadeaux et chèques cadeaux à votre site Wordpress. Créez vos offres, vendez et gérez vos ventes depuis une interface simple d'utilisation.

= Donnez à vos visiteurs la meilleure expérience d'achat de bons cadeaux =

Quelques clics suffisent pour choisir, personnaliser et acheter un cadeau !

== Installation ==

- Téléchargez le plugin et installez le sur votre site Wordpress
- Créez ou connectez le plugin à votre compte Ubiliz
- Créez vos offres si ce n'est pas encore fait
- Intégrez le shortcode [Ubiliz] sur la page où vous souhaitez voir apparaitre la liste de vos offres
- Le tour est joué, vous pouvez maintenant vendre vos bons cadeaux en ligne
